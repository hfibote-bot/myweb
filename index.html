关于这个站

  • 目的：练习 GitHub Pages 的发布流程
  • 形式：静态站点（不依赖服务器）
  • 内容：一段说明 + 一点点交互 + 小作品入口

你可以随便改文字、换配色、加图片、加更多页面。静态网页最爽的就是：改完再 push 一下就更新。
